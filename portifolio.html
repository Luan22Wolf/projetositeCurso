Meu Portfólio

Um projeto que trabalhei e deu muita dor de cabeça por eu ser novo na área de desenvolvimento, mas fiz com bastante atenção para que fosse algo bem seguro foi um sistema de sortear rifas.

O código faz um sorteio com números de um a cem, onde cada número está relacionado a um nome que estão armazenados em uma variável que trabalha com um dicionário.



Utilizei as anotações que fiz no curso para criar um gerador de números aleatórios usando a biblioteca random.


Estas são as bibliotecas usadas no sistema

Random time tkinter threading
A biblioteca `random` em Python oferece funcionalidades para gerar números aleatórios e realizar operações baseadas em aleatoriedade. A biblioteca time em Python permite manipular datas, horários e realizar operações relacionadas ao tempo. Tkinter é a biblioteca padrão de interface gráfica (GUI) para o Python, permitindo criar janelas, botões, entradas e outros elementos de interface. A biblioteca `threading` em Python permite a execução simultânea de várias partes do programa (threads), útil para tarefas concorrentes e paralelas.

# Função para realizar o sorteio e mostrar o resultado:

def sortear_numero():
global numero_sorteado
contador_label.config(text="Contagem regressiva: 5")
for i in range(5, 0, -1):
contador_label.config(text=f"Contagem regressiva: {i}")
time.sleep(1)
numero_sorteado = random.randint(1, 100)
nomes_vencedores = [nome for nome, numeros in rifa.items() if numero_sorteado in numeros]
if nomes_vencedores:
resultado = f"Número sorteado: {numero_sorteado}\nNome(s) do(s) vencedor(es): {', '.join(nomes_vencedores)}"
else:
resultado = f"Número sorteado: {numero_sorteado}\nNenhuma pessoa escolheu este número."
resultado_label.config(text=resultado)
